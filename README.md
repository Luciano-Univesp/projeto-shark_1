# Registro de Pedidos - Foodtruck

Aplicação web em Flask para registrar pedidos de um foodtruck.

## Como usar

1. Crie e ative um ambiente virtual Python.
2. Instale as dependências:

```bash
pip install -r requirements.txt
```

3. Execute o aplicativo:

```bash
python app.py
```

4. Abra o navegador em `http://127.0.0.1:5000`.

## O que faz

- Exibe um menu de itens do foodtruck.
- Permite selecionar item e quantidade.
- Calcula o total do pedido.
- Mantém o pedido em sessão até finalizar.
